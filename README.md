## DMIT 官方 TG 频道是什么情况

DMIT 目前运营的是**频道**，不是群组。

官方维护了两个频道：一个是中文运维日志频道（@DMIT_INC_CN），专门发中国用户关心的维护通知、故障公告、机房状态更新；另一个是官方主频道（@DMIT_INC），内容覆盖面更广，包括新产品上线、促销活动、全平台公告。

说一下区别：频道是单向推送，你只能接收消息，不能在里面发言。这和大多数商家搭的"群"不一样——你去 DMIT 官方渠道，看到的是运维通知和官方公告，不是用户聊天记录。

## 为什么找不到"DMIT 官方群"

说实话，DMIT 官方没有维护公开的聊天群。原因大概两点：一是 DMIT 的支持走的是工单系统，不靠群里回答问题；二是官方对渠道纯净度比较在意，他们明显不想自己的频道里充斥着用户讨论和灌水。

所以你搜到的那些"DMIT TG群"，基本都是非官方的——用户自发建的交流群，或者某些博主拉的讨论圈。这类群能不能用？聊得热闹，但里面的信息真假掺杂，偶尔还会有人在里面推别家的码。如果你想看 DMIT 的官方公告和机房状态，去官方频道是最直接的。
